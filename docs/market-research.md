## 1. 요약

한국 보험 시장은 약관·상품 spec·가격·인센티브의 **4중 정보비대칭** 위에 작동한다. 가입자는 보통 "월 보험료 + 진단비 액수" 두 개로 비교하지만, 실제 가치는 **총 누적 비용 + 약관 디테일 + 보험사 건전성 + 지급 신뢰성**의 4차원에서 결정된다. 그 결과 가입자의 약 60%는 의사결정에 회의적이며("경계 그룹"), 약 20%는 이미 호구 경험으로 어떤 보험 플랫폼도 불신("트라우마 그룹")한다.

설계사 채널은 전속에서 GA로 대이동 중이지만(전속 판매비중 21.5% → 14.4%), 거의 모든 인슈어테크 비교 앱(보맵·굿리치·시그널플래너)이 자체 GA를 운영해 사용자가 "객관 비교"라는 약속이 거짓임을 인지하는 순간 신뢰가 폭락하는 패턴이 반복된다. 빅테크(카카오페이·토스·네이버페이)는 자동차보험에서 비교·추천을 안착시켰지만 종신·건강·CI 등 장기 인보험에서는 사실상 진입하지 못했다.

한국 가입자가 "잘 가입했다"고 체감하는 4박자는 (1) 강매가 구조적으로 차단되어 있고, (2) 또래/시장 대비 내 위치가 보이며, (3) 보장의 구멍과 함정이 명시되어 있고, (4) 가입 후에도 결정의 정당성이 계속 확인되는 상태다. 글로벌·국내 통틀어 이 네 가지를 동시에 충족하는 서비스는 없으며, 특히 **(1) 강매 차단의 구조적 증명**이 가장 큰 빈자리다.

## 2. 시장의 구조적 비대칭

### 2.1 정보비대칭의 4중 구조

1. **약관 비대칭** — 약관이 금융·의학·법률 3중 용어로 구성. 보험사도 "쉽게 쓰면 송사 위험" 핑계로 개선 거부. 가입자가 약관을 끝까지 읽는 비율은 사실상 0%에 가까움.
2. **상품 spec 비대칭** — 같은 명칭의 보장이라도 보험사·상품마다 정의가 다름. 예: "뇌혈관 진단비"가 어떤 상품은 뇌출혈(I60–I62)만, 어떤 상품은 뇌혈관질환 전체(I60–I69)까지 포함.
3. **가격 비대칭** — 한국소비자원 조사: 동일 보장 순수보장형 암보험에서 보험사 간 가격 격차 남 21.4–61.9%, 여 21.1–49.2%. 같은 회사 안에서도 설계사가 짜는 방식에 따라 보험료가 2–3배 차이.
4. **인센티브 비대칭** — 1차년도 모집수수료 1200% 룰이 시행돼도 설계사 수당은 보험료에 비례. 진단비 키우고 환급형·갱신형으로 짤수록 수당이 늘어남 → 가입자 이익과 설계사 인센티브가 정면 충돌.

### 2.2 채널 구조 (2024 결산)

| 채널 | 손보 비중 | 13회차 유지율 | 25회차 유지율 | 특징 |
|---|---|---|---|---|
| GA 설계사 | 53.6% | 87.4% (최고) | 74.5% | 설계사 약 29만 명. 여러 보험사 비교 가능. 시책비 이동성 강함. |
| 방카슈랑스 | ~50% | 85.4% | 79.5% (최고) | 은행 신뢰 편승. 저축성·단순 상품 위주. |
| 전속설계사 | 7.2% | 85.7% | 72.5% | 약 18만 명. 본사 교육·관리 수준 높음. 단일사 상품만. |
| CM/다이렉트 | 19.2% | 85.0% | — | 자동차보험 위주. 종신·복잡 건강보장 사실상 불가. |
| TM | 적음 | 74.4% (최저) | — | 강매 인식 강함. |
| 빅테크 비교플랫폼 | 1–2% | — | — | 자동차 신규 위주. 종합 비교 불가. |

핵심 흐름: **전속 → GA 대이동** (전속 판매비중 2012년 21.5% → 2022년 14.4%). GA 이직 동기 1위는 수수료(24.4%), 2위 DB 제공(20.3%), 3위 정착지원금(13.9%). GA 설계사 1인당 월소득 478만 원으로 전속 304만 원의 1.57배.

규제 흐름: 1200% 룰이 2026년 7월부터 GA까지 확대 적용, 2029년부터 수수료 7년 분급. 모두 단기 갈아타기 권유(절판 마케팅)를 막기 위한 조치.

### 2.3 시장의 5대 트라우마 (학습된 불신)

1. **종신 = 저축 사기** — 생보 불완전판매 민원 중 종신보험 69.3%, 그중 10–20대가 36.9%.
2. **단기납 종신 절판 마케팅** (2023–2024) — "환급률 130%" 강조 후 중도 해지 시 손실. 금감원 소비자경보 발령.
3. **무해지환급형 함정** — 보험료 20–30% 싸 보이지만 중도 해지 시 환급금 0원. 약관대출 불가. 상반기 318만 건 판매 후 분쟁 폭증.
4. **유사암 함정** — 갑상선·제자리·경계성·기타피부암은 일반암 진단비의 10–20%만 지급. 가입자는 진단 후에야 알게 됨.
5. **갱신형 폭탄** — 40대 실손 3만 원 → 50대 6–8만 원 → 60대 10만 원 초과. 4세대 실손은 비급여 청구량에 따라 최대 +300% 할증.

부당승환(기존 해지 → 새 상품 갈아타기) 민원은 2026년 1분기 직전분기 대비 +54% 폭증.

## 3. 가입자 의사결정 5층 구조

가입자가 보험 상품을 평가할 때 다음 5개 층을 위에서 아래로 내려가며 따짐. 일반인은 보통 Layer 1–2만, 합리적 가입자는 Layer 5까지 본다.

### Layer 1: 보장 구조 (Coverage Architecture)

| 요소 | 일반인 판단 | 합리적 판단 | 함정 |
|---|---|---|---|
| 보장 항목 | 진단비 금액만 본다 | 진단비+통원+표적항암+후유증 조합 | 명칭 같아도 코드 범위 다름 |
| 보장 금액 | "5천만 원이면 충분" | 연봉의 1.2–2배 또는 연 생활비 3배 | 합산 한도 1–2.5억, 유사암은 10–20% |
| 보장 기간 | 100세가 좋다 | 90세 만기가 가성비 최적 (기대수명 86세) | 100세 만기는 보험료 30–50% 비쌈 |
| 면책/감액기간 | 모름 | 90일 면책 + 1–2년 50% 감액이 표준 | 89일째 진단 시 지급 거부 |
| 갱신형 vs 비갱신형 | 갱신형이 싸다 | 100세까지 누적은 갱신형이 8배 비쌈 | 초기 보험료만 강조됨 |
| 무해지/저해지 | 30% 싸서 좋다 | 7년 유지율 60%대 — 깨질 확률 매우 높음 | 중도 해지 시 환급 0원, 약관대출 불가 |
| 만기환급률 | 돌려받을 수 있으면 이득 | 차액을 적금에 넣는 게 더 나음 | 단기납 종신의 "환급률 130%" 마케팅 |

#### 갱신형 vs 비갱신형 사례 (40세 남성, 암진단금 5,000만 원)

- 갱신형: 초기 월 18,500원
- 비갱신형: 초기 월 71,150원 (3.85배)
- 100세까지 누적: 갱신형 약 1억 3,842만 원 vs 비갱신형 약 1,707만 원 (8배)
- 손익분기점: 약 67세

### Layer 2: 납입 구조

- 납입기간 20년 vs 30년: 월 보험료는 30년이 싸지만 총액은 30년이 약 300만 원 더 비쌈
- 갱신 시 보험료 인상: 1세대 실손 2026년 +3%, 2세대 +5%, 3세대 +16%, 4세대 +20%대. 4세대는 비급여 청구 많으면 다음해 최대 +300%
- 납입면제 특약: 특정 질병/장해 진단 시 잔여 보험료 면제 — 30년납이 면제 받을 기간 길어 유리할 수도

### Layer 3: 특약과 보장 코드 (가장 큰 정보비대칭)

| 항목 | 풀스펙 | 축소 스펙 | 차이 |
|---|---|---|---|
| 뇌혈관 진단비 | I60–I69 (뇌혈관질환 전체) | I60–I62 (뇌출혈만) | 뇌경색 진단 시 지급 여부 |
| 허혈성 심장질환 | I20–I25 | I21 (급성심근경색만) | 협심증 진단 시 지급 여부 |
| 일반암 vs 유사암 | 100% | 10–20% | 갑상선·제자리암 등 |
| CI 보험 | "중대한" 조건 충족 시 | 일반 진단으로는 X | 부지급률 일반 생보의 10배 (2013–2015) |

### Layer 4: 보험사·설계사 신뢰

#### 보험사 건전성 지표
- **K-ICS 비율** (舊 RBC, 지급여력비율): 감독기준 150% → 130%로 하향(2024). 7개 보험사 100% 미만 (2024)
- **신용등급** (AAA / AA+ 등)
- **보험금 부지급률** (손보협 공시)
- **금감원 분쟁조정 건수**

#### 설계사 신뢰 지표
- 자격증 위계: 보험설계사 < IFP < AFPK < CFP
- 공식 인증: 우수인증설계사(GA 약 2.2만 명), 골든펠로우(생보), 블루리본 컨설턴트(손보), GA 명장(약 4천 명)
- 블루리본 컨설턴트 데이터: 13회차 유지율 97.3% vs 업계 평균 86.7%, 25회차 94.0% vs 평균 69.6%, 불완전판매 0건

### Layer 5: 가격 합리성 휴리스틱

한국 가입자가 "비싸다/싸다"를 판단하는 5가지 휴리스틱:

1. **월 보험료 절대액** — 소득의 5–10%가 가이드. 연령대 평균 보험료 20대 약 18만, 30대 약 27만, 50대 정점 약 48만 (뱅크샐러드)
2. **동연령 평균 대비** — 비교 앱이 가장 강하게 자극하는 축. 시그널플래너 "같은 조건 최저가/최고가" 카톡 전송이 핵심 차별화
3. **보장당 단가** — "진단비 1억 원당 월 ○○원". 설계사 영업 멘트로 사용
4. **다른 보험사 견적 비교** — 자동차보험은 보편화, 종신·건강은 사실상 불가능
5. **보험가격지수** — 금감원 정의 객관 지표 (평균=100). 활용도 거의 0 (보험연구원 "정보 과부하" 진단)

## 4. 상품군별 결정변수

### 4.1 매트릭스 요약

| 상품군 | 핵심 spec | 가장 큰 함정 |
|---|---|---|
| 실손 | 세대(1–5세대) / 갱신주기 / 자기부담률 / 도수·MRI 한도 | 패키지 끼워팔기, 4세대 +300% 할증 |
| 종신/정기 | 보장기간 / 사망보험금 / 환급률 / 변액여부 | 저축 둔갑 판매, 단기납 절판 마케팅 |
| 건강(암/뇌/심) | 일반암 진단비 / 유사암 비율 / 뇌혈관 코드범위 / 갱신여부 / 재진단 특약 | 갑상선=10%, 뇌출혈만 보장 |
| CI/GI | 보장 질병 / "중대한" 조건 / 지급횟수 | 1회 지급 + 까다로운 조건, 부지급률 10배 |
| 어린이/태아 | 만기(30/100세) / 태아특약 / 산모 / 소아진단비 / 갱신형 특약 비율 | 22주 데드라인, 100세 만기 + 갱신형 폭탄 |
| 운전자 | 형사합의금 / 변호사선임비(자기부담률) / 벌금 / 가입시점 | 2026.1 개정 (변호사비 자기부담 50%) |
| 치아 | 면책기간 / 감액기간 / 임플란트 1개당 / 연 개수 | 치료 임박 가입 = 무의미 |
| 연금/변액 | 세제 / 공시·최저보증이율 / 사업비율 / 추가납입 한도 / 연금형태 | 5년 내 해지 = 원금 50% |
| 자동차 | 대인II/대물 한도 / 자상여부 / 무보험차상해 / 다이렉트 여부 | 매년 비교 안 함, 대물 2,000만 가입 |

### 4.2 상품군별 핵심 디테일

#### 실손의료보험
- 세대별 차이가 가장 큰 변수. 1세대(구실손)는 자기부담금 0, 4세대는 급여 20%/비급여 30%
- 4세대는 5년 재가입, 비급여 사용량에 따라 다음해 보험료 최대 +300%
- 2026.5 출시 5세대는 도수치료가 비중증 비급여로 분류돼 보장 제외
- 단독실손 의무화(2018)에도 어린이·치아 보험과 묶어 파는 "끼워팔기" 관행 잔존
- 1·2세대 → 4·5세대 전환은 비가역 (못 돌아옴)

#### 종신/정기 사망보험
- 동일 보장금액 시 종신이 정기 대비 보험료 2–5배
- 보험금액 산정: 연소득의 3–5배 또는 부양가족 수×생활비×남은 부양기간 + 부채
- 단기납 종신: 5·7·10년납 후 환급률 강조 → 저축으로 오인 판매. 금감원 민원 비중 2022 하반기 55.2%
- 변액종신: 펀드 수익률에 따라 보험금 변동. 사업비 후취로 5년 내 해지 시 환급률 50% 미만

#### 건강보험 (암/뇌/심)
- 진단비 권장: 일반암 3,000–5,000만 원이 시장 평균. 1억 이상은 보험료 부담 과대
- 유사암(갑상선·제자리·경계성·기타피부) 10–20%, 소액암(유방·자궁·전립선·방광) 10–30%
- 재진단암·2차암 특약: 재발·전이 시 일반암 100% 재지급 여부
- 뇌혈관 진단비 코드 범위가 결정적: I60–I62(좁음) vs I60–I69(넓음)

#### CI/GI 보험
- CI는 "중대한 암·중대한 뇌졸중·중대한 심근경색" — 일상생활 장해 잔존 등 조건 까다로움
- CI 1회 지급 후 종료. 다른 CI 질병 걸려도 못 받음
- GI(General Illness)는 "중대한" 조건 없앴지만 1회 지급에 비쌈
- 현행 일반 암보험·뇌·심 진단비가 CI 보장 범위를 더 넓게·여러 번·더 싸게 커버 → "CI 빼라"가 컨센서스

#### 어린이/태아보험
- 태아특약 가입 데드라인: 임신 22주 (12주 이내 권장 — 기형아 검사 전)
- 30세 만기 vs 100세 만기: 어릴 때 100세 가입 시 평생 동일 보험료 락인 가능, 단 갱신형 특약 비율 확인
- 산모 특약: 임신성 당뇨/임신중독증/제왕절개

#### 운전자보험
- 3대 핵심: 교통사고처리지원금(형사합의금), 변호사선임비, 운전자벌금
- 2026.1 개정: 신규 가입자는 변호사선임비 자기부담 50%, 한도 1,500–2,500만으로 축소. 기존 가입자는 약관 유지
- 자동차보험과 분리: 자보는 피해자 보상, 운전자보험은 가해자(=본인) 부담 비용 보장

#### 치아보험
- 면책기간: 가입 후 통상 90일 (질병 한정, 상해는 즉시)
- 감액기간: 보존치료 1년 / 보철 2년, 50% 지급
- 임플란트 1개당 보장금액: 보통 100–200만 원, 연 보장 개수 2–3개

#### 연금/저축/변액
- 세제: 연금저축 600만 + IRP 합산 900만 세액공제. 연금보험 비과세는 10년 유지·5년 이상 납입·월 150만 이내
- 사업비 후취: 초기 10–15% 차감 후 적립. 5년 내 해지 시 원금 손실
- 추가납입은 사업비 1–2%로 낮음 → "기본 적게, 추가납입 많이"가 정석
- 종신연금형 vs 확정연금형: 가입 후 변경 가능

#### 자동차보험
- 대인 II는 무한, 대물은 외제차 대비 3억–10억 권장
- 다이렉트가 설계사 채널 대비 평균 15% 저렴
- 할인 특약: 마일리지(최대 42%), 자녀할인(최대 24.4%), 블랙박스, UBI, 임산부, 티맵

## 5. 가입자 정보탐색 여정

### 5.1 6대 진입 트리거

자발적 인지보다는 외부 자극에서 시작:
1. 결혼/임신/출산
2. 취업/이직
3. 가족 발병/사망
4. 부모 보험 양도
5. 친구·지인 설계사 부탁
6. 기존 보험 갱신 인상 안내

### 5.2 탐색 funnel (2–4주)

| 단계 | 채널 |
|---|---|
| 1차 검색 | 네이버·구글 → 뱅크샐러드, 토스피드, KB Think, 시그널플래너 블로그 |
| 카페·커뮤니티 정성 탐색 | 임산부·엄마: 마미톡, 맘카페 / 사회초년생·남성: 클리앙, 디시 보험갤, 더쿠 리뷰판, 블라인드, 보배드림 / 30–40대 자산형: 짠돌이카페, 부동산스터디 |
| 유튜브 보강 | 보험왕(친근), 보험남자(가이드), 반값보험료(안티-보험사), 보험인사이트(폭로) |
| 비교 앱 시뮬레이션 | 굿리치, 보맵, 보닥, 시그널플래너, 보험클리닉, 보험다모아, 토스, 카카오페이 |
| 견적·상담 요청 | 카톡 비대면 강세. 시그널플래너 누적 250만 설치 |
| 최종 결정 | MZ도 80%가 결국 설계사 통해 가입 (KIRI) — "탐색-가입 갭" |

자동차보험 다이렉트 채널은 점유율 50.3%로 대면(49.7%) 첫 추월(2024). 생보·장기보장성은 비대면이 0.6%·6.2%로 미미.

### 5.3 세대별 차이

| 세대 | 가입률 | 평균 월 보험료 | 핵심 트리거 | 주 채널 |
|---|---|---|---|---|
| 20대 | 53% (가장 낮음) | 18–25만 | 첫 취업·독립, 부모 보험 양도, 지인 설계사 부탁 | 토스/카카오페이/디시/블라인드 |
| 30대 | 82% | 26–30만 | 결혼·임신·출산·내집마련 | 맘카페/재테크카페/유튜브 |
| 40대 | 88% (정점) | 39–40만 | 자녀 보장 확대, 부모 간병 우려 | 네이버 블로그/유튜브/설계사 |
| 50대 | 87% | 48만+ | 은퇴·연금·간병보험 | 대면 설계사 + 전화 |

KIRI 분석: MZ 만족도 최대 변수 = "합리성 + 자기주도성". "결정은 내가, 정보는 충분히, 영업은 최소" 구조가 PMF의 본질. 카톡 비대면 상담 4.8점의 시그널플래너가 이 공식.

### 5.4 여성 가입자 특수성

- 가족 보험 의사결정자 1순위 (자녀·배우자·시부모·친정 부모까지)
- 마미톡/맘카페의 결정력 압도적 — "OO보험사 OO설계사 강추" 식 실명 추천
- 태아·산모 특약 시한 압박: 임산부 70%가 12주 이전 가입 (마미톡 데이터)
- 우체국 "대한민국 엄마보험"(무료 공익) 인지도는 낮음

## 6. 페인포인트와 후회 패턴

### 6.1 가입 직전 막히는 5가지

1. **"이게 좋은 견적인지 판단 불가"** — 같은 보험사도 설계사마다 견적 다름
2. **약관 용어 장벽** — 금융+의학+법률 3중 용어
3. **갱신형 vs 비갱신형 트레이드오프**
4. **갈아탈지 유지할지** — 부당승환 민원 2026 1Q +54%
5. **거절 못하는 지인 영업** — 후회 1위 트리거. 클리앙·블라인드·82쿡에 "지인 보험 거절법" 글 폭증

### 6.2 가입 후 후회 패턴

| 패턴 | 사례/통계 |
|---|---|
| 저축인 줄 알고 종신 가입 | 생보 민원의 69.3%, 10–20대 36.9% |
| 무해지환급형 중도 해지 | 상반기 318만 건 판매 후 분쟁 폭증 |
| 유사암 함정 | 갑상선암 진단 후 일반암의 10%만 받음 |
| 갱신 폭탄 | 50대에 보험료 폭등 후 강제 해지 |
| 도수치료 거절 | 거절 분쟁 86.5%, 횟수 제한 56.8% |
| 모르고 들었다 | 27세 H씨 부모 보험 물려받아 월 30만 — 어떤 보험인지 모름 |

### 6.3 "내가 호갱이었나?" 트리거 3종

1. 친구가 같은 보장에 30–50% 싼 견적 자랑
2. 새 상품 출시 광고
3. 갱신 인상 통보

### 6.4 불완전판매 이후 행동

- **침묵 다수**: 도수치료 거절 사례 — 소송 비용·시간 부담돼 포기
- **민원·해지**: 종신보험 불완전판매 민원 2024 하반기 4,695건 중 종신 3,255건 (69.3%)
- **청약철회**: 증권 수령 15일 이내, 청약 30일 이내. 설계사 잠적으로 실행 못한 사례 다수
- **법적 처리**: 입증 시 기납입료 + 상사법정이자 6% 환급. 입증 자체가 어려움 — "말은 했는데 설계사가 안 적었어요"는 인정 안 됨, 전산 기록만 인정

### 6.5 금감원 / 협회 통계 (2025 상반기)

- 보험민원 28,137건 (전체 금융민원의 49%)
- 손보 21,452건 중 보험금 산정·지급 11,431건이 최다
- 생보 6,685건 중 보험모집 분쟁 2,236건
- GA 채널 비중: 손보 30.5%. GA 설계사 약 29만 명으로 전속 18만 명 추월

## 7. 신뢰 형성 메커니즘

### 7.1 강한 신뢰 신호

- 회사 브랜드 (삼성·교보·메리츠 등 대형사)
- 공식 인증 (우수인증설계사, 블루리본, CFP, GA 명장)
- 상담 시 "기존 가입내역 먼저 확인" — 갈아타기보다 보장 공백 보완 우선 제안
- 청구·AS 사례 포트폴리오 제시
- 무리한 권유 안 함, "지금 가입할 필요 없다"고도 말함
- 가족력·직업·과거 병력 디테일을 묻는 깊이

### 7.2 불신 신호 (즉시 이탈)

- "곧 단종", "이번 달까지만" — 절판 어휘
- 사은품·현금 제안
- 환급률만 강조, 사업비·해지 시 손실 미설명
- 카톡 답변이 늦거나 형식적
- 같은 청약서 템플릿을 누구에게나 적용

### 7.3 지인 추천의 위력과 부작용

지인·친척 설계사를 통한 가입은 한국에서 가장 흔한 진입 경로지만:
- 비교 부재 → 한 회사 상품만 들음
- 해지 시 인간관계 손상 우려로 손해 보고도 유지
- 설계사 이직·퇴사 시 AS 끊김 (전속설계사 13회차 정착률 51%대)
- 환수 룰을 모르고 가입자가 부담 느낌 (가입 2년 후엔 설계사 수수료 환수 거의 없음에도)

### 7.4 가입자 인지 세그먼트

| 세그먼트 | 비중 (추정) | 특징 |
|---|---|---|
| 트라우마 그룹 | ~20% | 이미 호구 경험. 어떤 보험 플랫폼도 안 믿음 |
| 경계 그룹 | ~60% | 의심하지만 도구·콘텐츠가 있으면 써봄. 휴대폰 번호 입력 직전에 약 50% 이탈 |
| 적극 비교파 | ~20% | 이미 여러 비교 앱 사용. 호갱 안 되려고 적극적 |

## 9. 가입자 만족 체감 메커니즘

벤치마크에서 추출한 패턴 라이브러리. 한국 시장에 적용 가능한 메커니즘 위주.

### 9.1 가격 합리성 시그널
- "동연령 평균 대비 -X%" 배지 (보맵, NerdWallet, 미국 ACA)
- "비교하지 않으면 평균 $4,914 손해" 같은 손실 프레임 카피 (NerdWallet)
- 보장당 단가 자동 산출 ("진단비 1억 원당 월 ○○원")
- "보험사 다이렉트와 동일 보험료" 명시 (토스 2025.4)

### 9.2 Coverage 최적성 시그널
- 카테고리별 "보장 부족/과보장" 진단 (보맵·시그널·해빗팩토리 리모델링)
- 30+ 정책 피처 비교 (GoCompare)
- "보장 누락 알람" 배너 — "3개 제안 모두 OOO 보장 없음"
- 30년 누적 보험료 그래프, 갱신/비갱신 손익분기점 표시
- 함정 자동 라벨 (무해지환급형, 면책기간, 유사암 10%, CI 1회 지급 등)

### 9.3 전문가 신뢰 시그널
- 사용자 언어 리뷰 태그 (닥터나우 "친절하게 설명해주셨어요(47)")
- 응답속도 라이브 SLA — "평균 응답 12분 / 7일 100% 응답"
- 자격·이력 자동 노출 (생/손보협 모집인 조회, CFP/블루리본/우수인증설계사 배지)
- 민원 이력 0건 뱃지 (최근 24개월)
- 정규직·기본급 상담사 정책 명시 (시그널플래너 "월 275만 기본급")

### 9.4 강매 차단 패턴
- "제안 받기 ≠ 콜백 동의"의 구조적 분리 — 제안은 앱 내 카드, 전화는 가입자가 "통화 요청" 버튼을 눌러야만 발생
- 결제 에스크로 (숨고페이)
- "이 제안 거절" 1탭 + 거절 사유 누적 → 매칭 ranking 학습

### 9.5 가입 후 합리화 / 만족 회수
- 가입 결정 요약 카드 (공유 가능 artifact)
- 3개월/1년 후 재진단 푸시
- 무사고/저활용 환급 (카카오페이손보 해외여행, 토스 걸음수)
- 무료 청구 컨설팅 (굿리치 보상청구팀)
- 숨은 보험금 찾기 (굿리치 850억)

### 9.6 진입 마찰 제거
- 본인인증 1회 → 모든 정보 자동 채움 (토스, Policygenius)
- 카톡 채팅 진입 (시그널플래너) / WeChat 임베디드 (WeSure)
- AI 1초 가설계 + 인간 상담사 fallback (시그널플래너, Lemonade Maya)

### 9.7 사용자가 떠나는 실패 패턴
- "받자마자 전화 폭탄" (PolicyBazaar, SelectQuote, 보맵 초기)
- "표준 견적 vs 실제 보험료 격차" (보험다모아 — 8년차에도 비판)
- "비교 후 보험사 사이트로 redirect" (The Zebra) — 정보 재입력 + 이탈
- "AI는 공감 못 한다" (Lemonade BBB 클레임 거절)
- "리뷰 조작 의심" (크몽 가이드 자체 명시)
- "자체 GA로 결국 판매" — 신뢰 한 번에 무너짐
- "개인정보 사고" (닥터나우/굿닥 과태료 5사)
